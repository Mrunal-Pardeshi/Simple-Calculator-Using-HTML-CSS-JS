# 🧮 Basic Calculator

Welcome to my **Basic Calculator** project! This is a simple calculator built with **HTML, CSS, and JavaScript** as part of my web development learning journey.

It can perform basic arithmetic operations through a clean and responsive user interface.

## 🚀 Live Demo

You can try the calculator here:

** https://mrunal-pardeshi.github.io/Simple-Calculator-Using-HTML-CSS-JS/**


## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear display button
- 📱 Responsive design for different screen sizes

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
